# mysql migration guide

Container should run to manipulate sql files in. Make sur you run `docker-compose up -d` before hand.

1. clean db and make it ready for importation
docker exec minecraft-db mysql -u root --password=root minecraft -e "DROP DATABASE minecraft; CREATE DATABASE minecraft"

2. Import sql file from host
docker exec -i minecraft-db mysql -u root --password=root minecraft < /home/bidetaggle/minecraft.sql


# import db inside container:

pv /minecraft.sql | mysql -u root --password=root minecraft

mysql -u root --password=root minecraft < /minecraft.sql
